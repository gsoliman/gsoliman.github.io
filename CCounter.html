CCounter- Android Application

The CCounter is a tool that allows you to count in sets of 1s, 2s, or 3s in a variety of situations. It can help in miscellaneous tasks and can even serve great religious use. The App version available for Android Devices and coming soon to other devices, but the browser version can be opened on any device.

Click begin, then enter a word that you would like to know how many times you say or sing, and start speaking/singing. It has a simple and easy to use interface.

It only works on Google Chrome

Languages and Tools Used

  • JavaScript
  • Google Script
  • HTML
  • CSS
  • WebSpeech API
