Realtime Voice Counter

This Web App alows users to enter a word they want to keep count of. Then it uses the microphone to detect when the word has been said or even sung. It can distinguish the word amongst other words and dialogue.

Click begin, then enter a word that you would like to know how many times you say or sing, and start speaking/singing. It has a simple and easy to use interface.

It only works on Google Chrome

Languages and Tools Used

  • JavaScript
  • Google Script
  • HTML
  • CSS
  • WebSpeech API
